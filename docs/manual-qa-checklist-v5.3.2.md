# SecretBase V5.3.2 人工验收清单

## 1. 启动、锁定与恢复

- [ ] Web、Windows、macOS 正常启动；本地服务不可用时显示“重新连接”，不会误显示为密码错误或空密码库。
- [ ] 解锁后快速切换全部条目、标签、密码组、回收站、备份和工具；在请求未完成时锁定，确认页面立即进入保护状态。
- [ ] 锁定后重新解锁，条目、标签、密码组、备份、同步主机、AI API Key、导入预览和恢复密码均不残留，旧请求不能覆盖新数据。
- [ ] 断网后重试条目、标签、密码组、备份和回收站加载；已有内容不被清空，恢复网络后重试成功。

## 2. 分页与列表交互

- [ ] 全部条目、更多标签、标签管理、密码组、已有条目选择、备份和回收站分别切换 5、10、20、50；列表完整显示，不出现超出数量后空白或隐藏。
- [ ] 在每个分页输入框填入空值、0、负数、501 和合法自定义值，确认自动回到合法范围；切换数量后回到第一页。
- [ ] 刷新或重新登录后确认各分页继续使用上次选择的数量。
- [ ] 回收站删除当前页最后一条，确认自动回到上一页；加载失败时底部不显示旧分页操作，并可重新加载。
- [ ] 备份列表加载失败时显示重试入口；创建备份、刷新列表和恢复过程中按钮不会重复提交。

## 3. 条目、标签与密码组

- [ ] 快速连续点击条目详情/编辑不同条目，确认最终显示最后一次选择的条目。
- [ ] 快速连续点击星标、批量星标、批量标签和删除，确认每项只提交一次，失败后选择仍保留。
- [ ] 密码组上移、下移和恢复默认排序时按钮进入处理中状态；排序失败后仍可再次操作。
- [ ] 从密码组详情新建条目，确认默认继承当前密码组；删除密码组后页面立即刷新并返回密码组视图。
- [ ] 标签管理的新建、编辑、批量删除和分页均可操作；失败时弹窗不误关闭。

## 4. 确认弹窗与导入导出

- [ ] 删除、批量操作、清空回收站、导出明文和同步危险操作失败时，确认弹窗保留并显示错误，可重试或取消。
- [ ] 导入加密备份或下载明文备份时取消主密码输入，确认原确认操作仍保持可处理状态；输入错误密码不会清空现有数据。
- [ ] 连续打开两个确认操作，确认旧操作完成后不会关闭或覆盖新确认弹窗。

## 5. AI 管家与专业工具

- [ ] AI 管家打开过程中立即关闭再重新打开，确认不会被旧响应重新填充或跳转。
- [ ] 专业工具请求期间切换模式、清空内容、关闭弹窗，确认控件按请求状态锁定且不会产生错位建议。
- [ ] 清除 AI 配置需要二次确认；失败后配置编辑界面和错误提示仍可操作。
- [ ] AI 发送预览前检查发送范围；确认普通模式不发送已有字段值，计划应用仍要求用户确认。
- [ ] AI 生成建议后快速切换会话，确认标题、条目详情和计划属于当前会话；应用失败后计划仍可重试。
- [ ] AI 设置页获取模型列表时关闭并重新打开页面，确认旧响应不会覆盖新设置，重新获取按钮不会永久停留在“获取中”。
- [ ] Android AI 设置清除成功、失败后重新打开，确认“专业工具”“AI 管家”和服务设置均可继续进入，不会显示残留的“请求处理中”。
- [ ] Android AI 管家或专业工具发起请求后点击“取消请求”，确认网络请求停止、输入和工具恢复可用；随后迟到响应不会生成建议、错误或覆盖新请求。

## 6. 同步与桌面更新

- [ ] 同步历史读取失败时点击“重新加载”可恢复；同步进行中关闭按钮不会造成页面永久锁定。
- [ ] Web/桌面加入同步空间时先测试 WebDAV，再直接点击“加入并同步”；即使未填恢复码按钮也不能无解释地置灰，点击后必须在弹窗内和页面顶层显示缺项提示，填写恢复码后可继续。
- [ ] 使用已有浏览器缓存打开站点，确认同步配置弹窗、错误 Toast 和配对解析来自本次资源版本，不需要手动清空缓存。
- [ ] Web/桌面创建空间后用 Android 扫码或配对链接加入，应用密码仍需手动输入；锁定后配对材料和密码清空。
- [ ] Windows/macOS 手动检查更新、下载、取消和安装失败后仍保留可操作重试入口；当前无正式 Release 时显示中性提示。
- [ ] Windows/macOS 更新接口响应慢于轮询间隔时，确认不会出现重复请求、进度跳回或旧状态覆盖新状态。
- [ ] Windows 托盘隐藏、托盘重新打开和完全退出均要求符合设置的锁定/确认策略；关闭确认失败后不会卡死。
- [ ] 在设置页依次打开修改主密码、备份中心、导入预览/冲突和桌面状态，按 Escape 确认先关闭最上层弹窗，底层设置仍保留。

## 7. Android

- [ ] Android 前后台切换、系统锁屏和指纹解锁反复执行，确认不会出现半屏白屏、底部导航残留或保护遮罩卡死。
- [ ] Android 同步连接测试成功后，“加入并同步”保持可操作；缺少恢复码、HTTPS 地址或 WebDAV 凭据时，弹窗内保留错误并通过顶部 SnackBar 提示，不出现无解释的灰色按钮。
- [ ] 指纹凭据失效时只提示重新开启生物识别，不暴露 Vault 内容；主密码解锁仍可用。
- [ ] Android 分页、字体模式、条目/标签/密码组来源返回、AI `+` 面板和连续返回退出均正常。
- [ ] 从密码组或标签进入条目后切换到设置/管家，再按返回；确认先回到保留筛选的条目页，下一次返回才回到来源页。
- [ ] Android 更新在断网、超时、切后台后可重试或续传；成功覆盖安装后 Vault、偏好和生物识别状态保持。

## 8. 本轮新增交互回归

- [ ] 移动同步加入、立即同步、冲突解决和历史恢复成功后，模拟本地状态读取失败；确认提示“操作已完成”并出现重新读取入口，不会要求重复执行远端操作。
- [ ] 移动同步压缩历史或轮换密钥后，恢复码读取失败时确认旧恢复码不会继续显示，页面保留警告并允许重新读取。
- [ ] 移动 AI 在发送确认页点击取消，确认不会发出网络请求；生成建议后点击“放弃建议”，确认建议和待处理请求都被清除，密码库没有变化。
- [ ] 移动专业工具生成建议后尝试切换工具、打开 AI 设置、修改输入和重复生成，确认这些入口被锁定；处理或放弃建议后恢复可用。
- [ ] 移动 AI 操作失败时确认已有对话仍可见；删除当前对话后确认自动回到新会话，下一次发送不会使用已删除的会话 ID。
- [ ] Web/桌面在回收站恢复或彻底删除、标签保存/合并、备份恢复、桌面更新下载期间尝试关闭和翻页，确认控件禁用且操作完成后不会出现旧数据覆盖。

## 9. 发布边界

- [ ] 正式 Release 生成后，核对三端资产、更新清单、签名文件和 SHA-256 校验和均来自同一个 `v5.3.2` 标签构建。
- [ ] 发布前确认仓库中没有 `.env`、Vault、备份、日志、WebDAV 凭据、服务器路径或私人订阅地址；只发布构建产物和签名清单。
